# ほうにゃんスタンプ台帳

特別支援のプリント取り組みを、児童ごとにスタンプシートで管理するローカルWebアプリです。

## 使い方

1. `index.html` をブラウザで開きます。
2. 先生ページで児童を登録します。
3. 児童ページで児童を選び、「スタンプをおす」から押すスタンプと個数を選びます。
4. 先生ページで履歴の取消、報酬交換、バックアップを管理します。
5. 先生ページの「組の管理」で組を追加し、「時間割」と「カレンダー」を設定します。

## 保存

- データはこのPCのブラウザ内に保存されます。
- 別PCへ移すときや念のため残したいときは、先生ページからJSONを書き出してください。
- 復元するときは、同じ先生ページからJSONを読み込みます。

## v1でできること

- 児童ページと先生ページの分離
- 児童ページでのスタンプ追加と20個スタンプシート表示
- プレビューで複数スタンプを選んで一括追加
- 先生ページでの児童追加、編集、削除、スタンプ取消
- 今までに集めたスタンプと使えるシートの分離
- 累計20/40/60個でのスタンプ解放
- 使えるシート1枚単位での報酬交換
- 着せ替え交換枠の使用不可表示
- 紙にスタンプを押すような押印アニメーション
- 20個目で出るほうにゃんのシート完成アニメーション
- 新しいスタンプが使えるようになった時のほうにゃん解放アニメーション

## カレンダー・時間割

- 組の追加、編集、有効・無効、並び替え、完全削除
- 組ごとの月曜から金曜・最大10時間の基本時間割
- 教科の追加、編集、有効・無効
- 時間割の適用期間、別の組へのコピー、初期化
- 月間カレンダー、学校全体または組別の予定、連続日・毎週の予定
- 休業日、土曜授業、短縮授業、行事日の登校日判定
- 日付ごとの一部時間割変更と、基本時間割へ戻す操作

予定と時間割は、既存のJSONバックアップに一緒に保存されます。日本の祝日は端末内で判定し、学校予定の「登校日への影響」で上書きできます。

## ミッション連携の土台

`window.HounyanSchoolCalendar` から、今後のミッション機能で使う次の処理を参照できます。

- `isSchoolDay(date, groupId)`
- `schoolDaysAfter(date, count, groupId)`
- `countSchoolDaysBetween(startDate, endDate, groupId)`
- `getDayTimetable(groupId, date)`
- `hasSubjectOnDate(groupId, date, subjectId)`
- `canGenerateDailyMission(studentId, date)`

児童の「テスト児童として使用する」を有効にすると、ミッション生成可否の判定では土日祝・休業日を通過します。現時点ではミッション画面そのものは未実装なので、この設定は次のミッション実装で利用されます。
